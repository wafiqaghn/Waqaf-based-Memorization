## 📖 Overview

Tartila is a modern Quran learning platform designed to improve Quran memorization through the **Waqaf-Based Memorization Method**. Instead of memorizing verse by verse only, Tartila guides users to memorize according to natural stopping points (waqaf), making memorization more structured and meaningful.

This project was developed as a capstone project and aims to combine Islamic learning with modern web technologies and AI-powered personalization.

---

## ✨ Features

- 📖 Quran reader
- 🧠 Waqaf-based memorization
- 🎯 Memorization progress tracking
- 🔍 Search verses
- 🌙 Dark mode
- 🌍 Multi-language support
- 📱 Responsive interface

## 🚀 Getting Started

### Requirements

- Node.js 20+
- Corepack

```bash
corepack enable
```

### Installation

```bash
git clone https://github.com/wafiqaghn/Waqaf-based-Memorization.git
cd Waqaf-based-Memorization

pnpm install
pnpm dev
```

Open:

```
http://localhost:3000
```
